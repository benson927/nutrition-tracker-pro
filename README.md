# 🚀 Nutrition Tracker Pro

一款基於 React 打造的極致美學健康追蹤應用。採用前衛的 **Liquid Glass (液態玻璃)** 視覺風格，提供直觀的 BMR 計算、飲食追蹤與社群互動功能。

---

## 🛠 核心技術棧 (Tech Stack)

- **Frontend**: React.js, TypeScript
- **Styling**: Tailwind CSS (v4)
- **UI Framework**: Liquid Glass System (Custom Vanilla CSS)
- **State Management**: React Hooks (useState, useEffect, useMemo)
- **Data Visualization**: Recharts (Modern monotone curves)
- **Validation & UI Components**: SweetAlert2, Lucide React Icons
- **Backend & Database**: FastAPI (Python), SQLite3

---

## 🚀 開發實錄 (Development Roadmap)

本專案經歷了五個關鍵的開發階段，從視覺探索延伸至功能與安全的極致追求：

- **Phase 1: Visual Exploration**
  探索 3D 視覺效果與慣性輪播系統，為專案確立了高品質的動態視覺基礎。

- **Phase 2: Liquid Glass Era**
  正式定調以「純黑背景 (#000000)」搭配「流體呼吸感光暈」的 Liquid Glass 玻璃擬態美學，營造深邃且具未來感的介面。

- **Phase 3: Performance Pivot**
  執行系統瘦身，移除冗餘的三方模組，將精力集中於核心 TDEE 計算邏輯與資料安全機制，提升整體運行效能。

- **Phase 4: UI Refinement**
  實作 3x2 / 2x2 對稱佈局與純 CSS 驅動的微互動效果。此階段優化了 Dashboard 的平衡感，並引入了半透明微型滾動條設計。

- **Phase 5: Logic & Security**
  完善論壇（Forum）的資料歸屬邏輯，實作發文身分「凍結」機制。同時建立全域登出（Global Logout）流程，確保使用者資料與隱私獲得最嚴密的保護。

## ✨ 專案特點 (Core Features)

- **完美的 Liquid Glass 質感**：深層的模糊半透明卡片、極細高光邊框與流體呼吸背景。
- **智慧數據追蹤**：反應式熱量曲線圖表，自動計算 TDEE 與剩餘可攝取量。
- **嚴謹資料管理**：具備強大的資料驗證機制，並確保論壇內容的署名一致性。
- **優雅的互動體驗**：全站採用輕量化 CSS 動畫，確保操作過程中的視覺連貫與絲滑感。
